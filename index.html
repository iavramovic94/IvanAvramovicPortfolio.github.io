Projects I've Worked On

A collection of portfolio projects I've worked on in SQL, Tableau, and Python

Data Cleaning

Looking at Nashville Housing Data, I cleaned up the initial dataset, first by standardizing the date format, populating property address data, as well as getting rid of null property addresses. Next, I broke out the addresses into individual columns (address, city, state), split up city and state fields and changed the Y/N sold fields to Yes and No. Lastly, I removed all duplicates and deleted unused columns.

COVID-19 SQL Project

This project explores the global COVID-19 Database querying things like percentage of population infected, which countries had the highest infection rates compared to their populations, and looking at total death percentage. All queries for this project are in the github folder

Pharma Sales Data Exploration

I observed sales data which consisted of 60,000 transactions collected from 2014 to 2019 to observe patterns like hourly, weekly, daily, and monthly sales trends. The visualizations for this projects are within the Tableau Visualizations project link.

Tableau Visualizations

A compilation of Tableau visualizations I created for various projects listed on this page, including some dashboards as well.

EU Covid Data Exploration

Working off the same dataset as the Global COVID-19 project, I looked at the differences between EU and non EU countries when it comes to their percentage of population infected, as well as death percentages pre and post vaccine availability.
